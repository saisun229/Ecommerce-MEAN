(1) Make sure Node and CLI are installed in your computer.

(2) Development computer in a Windows 7 (64 bit) laptop. This project was built and tested in this computer that has the versions as listed in Appendix-A. 

(3) Remote Sandbox server is a Windows 2012 Server (64 bit): Author tested the zip file application whose versions are listed in Appendix-B.

(4) Unzip the .zip file

(5)  In the command prompt, go to the unzipped directory. Run the command "npm install". Ignore the warnings. Will take about 5 minutes to complete.

(6)  In the command prompt, go to the unzipped directory. Run the command "ng serve --port 4200" to start the Angular 2 project.

(7)  Now you should be able to use the Angular 2 app in the browser thru the URL http://localhost:4200/
